# projek-mikrokontroler-scanuangtcs3200
Projek Mata Kuliah Mikrokontroler &amp; Mikroproseesor menggunakan sensor tcs3200 yang membuat scan uang kertas pecahan rupiah

*Perlu diperhatikan pada projek saya ini tidak 100% langsung bisa berhasil semua pecahan uang rupiah nya. Karena beberapa faktor yaitu cahaya dan pecahan setiap uang kertas itu tidak dalam satu warna yang utuh seperti origami. Pada saat menscan uang nya saya mengkalibarsi data rgbnya menggunakan satu titik yang warnanya terlihat tampak jelas.*
